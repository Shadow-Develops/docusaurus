# CDN Manager

This is the official documentation for [CDN Manager](https://shadowdevs.com/store/cdnmanager). Here you can find information on how to install, configure, and manage your CDN Manager instance.

## What is CDN Manager?

CDN Manager is a powerful, full-featured web-based content delivery and file management system designed for organizations that need advanced media organization, granular permissions, and robust access control. Built with modern web technologies, it provides a secure and scalable solution for managing and delivering your files.
