# Global Product Layout

The Global Product Layout feature allows you to customize the default layout and sections for all product pages. This provides a consistent structure across your store while still allowing per-product customization.

## Accessing Global Layout

Navigate to **Admin Panel > Pages > Products** to manage the global product layout.

## Section Types

The same section types available for individual products:

- **Product Description**: Main product details
- **Related Products**: Similar products grid
- **Product Features**: Feature highlights
- **Additional Info**: Specifications and details
- **FAQ**: Frequently asked questions
- **CTA Banner**: Call-to-action section
- **Testimonials**: Customer testimonials
- **Reviews**: Product reviews
- **Text Content**: Custom markdown content

## Managing Global Layout

### Adding Sections

1. Click desired section type in sidebar
2. Section added to bottom of layout
3. Configure section details
4. Save changes

### Reordering Sections

1. Click and drag section by move icon
2. Drop in desired position
3. Save to persist order

### Configuring Sections

1. Expand section
2. Fill in section-specific fields
3. Add items if applicable
4. Save changes

### Enabling/Disabling Sections

Toggle sections on/off without deleting them.

### Deleting Sections

Remove sections permanently from global layout.

## Per-Product Overrides

Individual products can override the global layout by adding their own sections in the product editor.

## Permissions

- `owner` - Full layout management
- `managePages` - Edit global layout
