# Settings

Store Settings control global store behavior, appearance, filtering options, and special features like tags and giftcards. This centralized configuration affects the entire store.

## Accessing Store Settings

Navigate to **Admin Panel > Products > Settings** to configure store settings.

## General Settings

### Sales Tax

Configure tax or fees added to purchases.

**Options:**

- Fixed amount: `5.00`
- Percentage: `7.5%`
- Tax label text: "Tax & Fees" (customizable)

### Demo Link Button Text

Customize the button text for product demo links.

**Default:** "View Demo"

### Store Search Bar

Enable/disable the product search functionality.

**Options:**

- True: Search bar visible
- False: Search bar hidden

### Price on Image

Toggle price display on product thumbnails.

**Options:**

- True: Price shown on image
- False: Price shown below image

## Product Filtering

Configure which filters appear in the store:

**Available Filters:**

- **Category**: Filter by product categories
- **Price**: Filter by price range
- **Tags**: Filter by product tags
- **Type**: Filter by product type (paid, free, subscription, link)

Enable or disable each filter independently.

## Product Tags

Create and manage product tags for organization and filtering.

### Creating Tags

1. Click "Create Tag" card
2. Enter tag name
3. Save tag

Tags become available for product assignment.

### Deleting Tags

1. Find tag in tags list
2. Click "Delete"
3. Confirm deletion

**Note:** Deleting a tag removes it from all assigned products.

## Giftcards

Create purchasable giftcard products.

### Creating Giftcards

1. Click "Create Giftcard" card
2. Configure giftcard:
   - **Amount**: Giftcard value
   - **Name**: Product name
   - **Description**: Product description
3. Save giftcard

Giftcards function as special products that credit customer accounts.

## Permissions

- `owner` - Full settings access
- `manageProducts` - Modify store settings
